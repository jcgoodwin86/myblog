# Go Blog Project

An in-progress blog server using Go's standard library and templating.

## Stack
- HTTP: net/http (standard library)
- Templating: Templ (type-safe Go templates, .templ files)
- Markdown rendering: Goldmark
- Content: Markdown files (.md)
- Status: Work in progress

## Architecture
net/http → Templ → Goldmark → Markdown files
